📊 About BetRivals

BetRivals is a comprehensive football analytics platform developed as a database management project. We combine multiple data sources to provide deep insights into player performance, team statistics, and match outcomes through a carefully designed relational database system.

🎯 Project Overview

Our platform integrates seven interconnected databases with robust foreign key relationships, enabling complex queries and comprehensive football analysis. Each team member has taken ownership of specific datasets, ensuring data quality and expertise in their respective domains.

🔗 Database Architecture

Our relational database system features carefully designed foreign key relationships that ensure data integrity and enable powerful cross-table queries:

  • Player-Match Integration: Links player performance data with specific matches and teams
  • Team Relationships: Central teams table referenced by season stats, match data, and player records
  • Temporal Tracking: Year and season fields enable historical analysis and trend identification
  • Match Analytics: Connected match_data and match_info tables provide comprehensive match insights
  • Shot-Level Detail: Granular shot data linked to players, matches, and teams for advanced analytics

📈 Key Features

BetRivals provides advanced football analytics through:

  • Expected Goals (xG): Statistical measure of shot quality across players, teams, and matches
  • PPDA Analysis: Passes Per Defensive Action metrics to evaluate pressing intensity
  • FIFA Integration: Combines real-world performance with FIFA ratings for holistic player evaluation
  • Shot Maps: Spatial analysis of shot locations and quality
  • Season Tracking: Historical performance data spanning multiple seasons
  • Multi-dimensional Analysis: Cross-reference players, teams, matches, and seasons seamlessly

🚀 Technologies & Implementation

The platform is built using Flask as the web framework, with a normalized relational database design following best practices in database management. Our implementation emphasizes data integrity through primary and foreign key constraints, efficient query performance, and scalable architecture.

💡 Project Goals

Through BetRivals, we aim to demonstrate proficiency in database design, implementation, and management while creating a practical application that football enthusiasts and analysts can use to gain deeper insights into the beautiful game. Each team member's contribution ensures comprehensive coverage of football analytics domains.
